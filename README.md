### 구현 기능

- 작업 목록 읽기
- 마우스나 키보드로 작업 추가하기
- 마우스나 키보드를 사용하여 작업을 완료로 표시하기
- 마우스나 키보드를 사용하여 작업 삭제하기
- 마우스나 키보드를 사용하여 작업 편집하기
- 작업을 특정한 집합으로 나누어 보기: 전체 작업, 진행 중인 작업, 혹은 완료된 작업들.
